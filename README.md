# 🍎 FruitKart – Offline Billing App

FruitKart is a lightweight **offline billing web app** designed for small fruit shops.
It works fully on mobile and can be converted into an **Android APK**.

---

## 🚀 Features

### 🧾 Billing System

* Generate bills instantly
* Clean **table-format bill output**
* Shows total price automatically

### ⚖️ 3 Smart Modes

* **Weight Mode** → Enter quantity in kg
* **Gram Mode** → Enter weight in grams
* **Budget Mode** → Enter money, app calculates quantity

### 🧠 Smart Calculations

* Auto conversion:

  * ₹ → kg
  * kg → grams
* Accurate pricing based on mode

### 🗂️ Fruit Management

* Add unlimited fruits
* Update prices anytime
* Data saved locally

### 📜 Bill History

* Stores previous bills (24 hours)
* View all past bills
* Delete all bills option with in-app popup

### 📱 Mobile UI

* Full-screen Android-style layout
* Bottom navigation (Setup / Billing / History)
* Floating add button (+)
* Sticky “Generate Bill” button

### 🔒 Offline Support

* Works without internet
* Uses **localStorage**
* Perfect for shop usage

## ⚙️ How to Run

### 💻 On PC

1. Download project
2. Open `index.html` in browser

---

### 📱 Convert to APK

#### Method 1: Using GoNative

1. Upload project to hosting (GitHub Pages / Netlify)
2. Copy website URL
3. Paste in GoNative
4. Build APK
